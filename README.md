# Mangalam HDPE Pipes - Landing Page

A responsive landing page for Mangalam HDPE Pipes, a manufacturer of premium HDPE pipes and fittings for modern infrastructure. The page showcases product features, technical specifications, and provides interactive elements for user engagement.

## Description

This project is a single-page website built with vanilla HTML, CSS, and JavaScript. It features a modern, professional design with smooth animations, responsive layout, and interactive components. The website is designed to work seamlessly across desktop, tablet, and mobile devices.

## Technologies Used

- HTML5 - Semantic markup and structure
- CSS3 - Styling with Flexbox and Grid layouts
- JavaScript (ES6) - Interactive functionality and DOM manipulation
- No external frameworks or libraries

## Features Implemented

### Navigation
- Fixed header with logo and navigation menu
- Sticky header that appears on scroll with smooth animation
- Mobile-responsive navigation with hamburger menu

### Hero Section
- Image carousel with automatic rotation (4-second intervals)
- Manual navigation with previous/next buttons
- Thumbnail navigation for quick slide selection
- Hover zoom effect on carousel images
- Pause auto-rotation on hover
- Price display with delivery information
- Call-to-action buttons for quotes and technical specifications

### Content Sections
- Trusted companies logo carousel with infinite scroll
- Technical specifications table with dark theme
- Feature cards grid (6 cards) with hover effects
- FAQ accordion (expand/collapse functionality)
- Applications carousel with navigation controls
- Manufacturing process tabs with content switching
- Customer testimonials with horizontal scroll
- Product showcase with image cards
- Resources section with downloadable materials

### Contact
- Contact form with validation
- CTA section with blue background
- Footer with company information and links

### Interactive Elements
- Smooth scroll navigation
- Hover animations on cards and buttons
- Fade-in animations as elements enter viewport
- Form submission handling
- Download button functionality

### Responsive Design
- Desktop layout (1440px with 100px padding)
- Tablet breakpoint (1024px)
- Mobile breakpoint (768px)
- Mobile-first approach

## How to Run the Project Locally

1. Download or clone the project files to your local machine.

2. Ensure all three files are in the same directory:
   - index.html
   - styles.css
   - script.js

3. Open the `index.html` file in a modern web browser (Chrome, Firefox, Safari, or Edge).

4. The website will load and all interactive features will be functional.

No server setup or build process is required. The project runs entirely in the browser.

## Customization

To customize the website:

1. Update colors in `styles.css` by modifying CSS custom properties in the `:root` selector
2. Replace placeholder images with actual product images
3. Update text content in `index.html`
4. Modify animation timings and transitions in `styles.css`
5. Adjust breakpoints for responsive design as needed
